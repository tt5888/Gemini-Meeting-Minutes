# Gemini-Meeting-Minutes
## 這個專案使用gemini-2.0-flash，辨識音檔後產生兩份文件，產生逐字稿與會議摘要。

逐字稿會辨別不同的使用者，並標注 Speaker A 與 Speaker Ｂ，並標時間戳記。

會議摘要會產生，會議主題、討論議題與會議執行項目。
